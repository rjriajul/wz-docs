# Commands

This is where you learn everything the bot can do. Pick a category below.

**Need quick help?** Send just the command name to the bot without any arguments — it will reply with detailed usage. For example, just `/mirror` shows all mirror options.

## Categories

| Category | What you can do |
|---|---|
| [Basic](./basic) | `/start`, `/help`, `/ping`, `/login` — getting started |
| [Mirror](./mirror) | Download files to Google Drive, Rclone, or cloud storage |
| [Leech](./leech) | Download files and send them to your Telegram chat |
| [Uphoster (DDL)](./upload) | Upload files to file hosting sites |
| [File Management](./manage) | Check progress, cancel tasks, get file info |
| [Search](./search) | Find torrents, Google Drive files, IMDB info, anime, dramas, Usenet |
| [Settings & RSS](./settings) | Personal settings, categories, and automatic RSS downloads |
| [Admin](./admin) | Owner and sudo commands for bot management, broadcasting, plugins |
| [Arguments](./arguments) | Full reference of every argument and where it works |
