# Uphoster (DDL) Commands

## `/uphoster [link/file]`

**What it does:** Uploads a file to Direct Download (DDL) hosting sites like Gofile, Streamtape, etc.

**How to use:**
```
/uphoster https://example.com/file.zip
```

You can reply to a file or provide a link. The bot will upload it to supported DDL hosts and give you the download links.

**Who can use it:** All authorized users.

**Supported hosts:** Gofile, Streamtape, and other configured DDL services.
