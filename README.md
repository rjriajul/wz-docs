# wz-docs

Documentation for [WZML-X](https://github.com/SilentDemonSD/WZML-X) — a Telegram mirror and leech bot.

Built with [VitePress](https://vitepress.dev/).

## Local dev

```bash
npm install
npm run docs:dev
```

## Build

```bash
npm run docs:build
```

Preview with `npm run docs:preview`.
